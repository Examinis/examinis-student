# Examinis Student

The Examinis Student module is a Ruby on Rails application designed to manage the student module. In this module, a student will be able to create an account, authenticate, and take exams. Additionally, students can receive custom feedback generated by a Language Model (LLM) on their responses.

## Running the Rails Application

1. **Clone the Repository**
    ```sh
    # using SSH
    git clone git@github.com:Examinis/examinis-student.git
    # using HTTPS
    git clone https://github.com/Examinis/examinis-student.git
    ```

2. Install Docker Desktop on your machine according to your operating system.  
   Download link: [Docker Home and Download Page](https://www.docker.com/)

3. After installing Docker and cloning the project, navigate to the project's root directory:
    ```sh
    cd examinis-student
    ```

4. Run Docker Compose to build and start the Rails and PostgreSQL containers:
    ```sh
    docker compose up
    ```

5. In another terminal window, create the database specified in the `config/database.yml` file:
    ```sh
    docker compose run web rake db:create
    ```

6. Access the application in your browser at: [http://0.0.0.0:3000](http://0.0.0.0:3000).
